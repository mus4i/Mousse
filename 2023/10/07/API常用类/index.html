char字符数组转换成String的一种方式

    '''java
        char [] arr = new char[]{'a', 'b', 'c'};
        String str = new String(arr);
        System.out.println(str);
    '''

String类常用的方法：

  1. public int length() 获取字符串的长度返回(就是字符个数

     '''java
     System.out.println(str.length());
 '''

  2. public char charAt(int index) 获取某个索引位置处的字符返回 

     '''java
     for (int i = 0; i < str.length(); i++) {
                 char ch = str.charAt(i);
                 System.out.println(ch);
             }
 '''

  3. public char[] toCharArray(): 将当前字符串转换成字符数组返回

     '''java
     char[] chars = str.toCharArray();
 '''

  4. public boolean equals(Object anObject) 判断当前字符串与另一个字符串的内容一样，一样返回true

     '''java
     System.out.println(str.equals(str2));
 '''

  5. public boolean equalsIgnoreCase(String anotherString) 判断当前字符串与另一个字符串的内容是否一样(忽略大小写)

     '''java
     System.out.println(str.equalsIgnoreCase(str2));
 '''

  6. public String substring(int beginIndex, int endIndex) 根据开始和结束索引进行截取，得到新的字符串(包前不包后)

     '''java
     String rar = "学java死路一条";
             System.out.println(rar.substring(0, 5));
 '''

  7. public String substring(int beginIndex) 从传入的索引处截取，截取到末尾，得到新的字符串返回

     '''java
     System.out.println(rar.substring(5));
 '''

  8. public String replace(CharSequence target，CharSequence replacement) 使用新值，将字符串中的旧值替换，得到新的字符串

     '''java
     System.out.println(rar.replace("学", "卷"));
 '''

  9. public boolean contains(CharSequence s) 判断字符串中是否包含了某个字符串

     '''java
     System.out.println(rar.contains("java"));
 '''

  10. public boolean startswith(string prefix) 判断字符串是否以某个字符串内容开头，开头返回true，反之

    '''java
    System.out.println(rar.startsWith("学java"));
'''

  11. public String[] split(string regex) 把字符串按照某个字符串内容分割，并返回字符串数组回来 

    '''java
    String string = "呵呵哒，呵呵哒，呵呵哒";
            String[] heheda = string.split("，"); 
            for (int i = 0; i < heheda.length; i++) {
                System.out.println(heheda[i]);
            }
'''

ArrayList类的常用方法

1.

add(E e) 将指定元素添加到集合的末尾

    '''Java
        ArrayList list = new ArrayList();
        list.add("呵呵哒");
        list.add(123);
        list.add(1.22);
        System.out.println(list);
    '''

2.
add(int i, E e) 往集合中的某个索引位置添加元素

    '''Java
        list.add(1, "heheda");
        list.add(1,"呵");
        System.out.println(list);
    '''

3.
get(int i) 获取指定索引位置的值

    '''java
        System.out.println(list.get(1));
    '''

4.
size() 返回集合中的元素数量

    '''java
        System.out.println(list.size());
    '''

5.
remove(int i) 删除指定索引位置的元素， 并返回删除的值

    '''java
        System.out.println(list.remove(1)); 
    '''

6.
remove(Object j) 删除指定的元素， 并返回删除是否成功

    '''java
        System.out.println(list.remove("呵"));  //默认删除第一个出现的指定元素
    '''

7.
set(int i, Element e) 替换指定索引位置的元素, 并返回被修改的值

    '''java
        System.out.println(list.set(1, "呵呵哒"));
    '''

包装类的一些方法

Objects类的常用方法

equals(Object a, Object b) 判断两个对象内容是否相等

    '''java
        String str = "heheda";
        String str2 = "heheda";
        System.out.println(str.equals(str2));  //原来这种方法有安全问题， 可能报空指针异常
        System.out.println(Objects.equals(str, str2));
    '''

isNull(Object obj) 判断一个对象是否是null 

    '''java
            String str = null;
            String str2 = "heheda";
            // System.out.println(str.equals(str2));
            System.out.println(Objects.equals(str, str2));

            System.out.println(Objects.isNull(str));
            System.out.println(Objects.isNull(str2));
    '''

nonNull(Object obj) 判断一个对象是否不是null

    '''java
        System.out.println(Objects.nonNull(str2));
    '''

System.exits()
Ststem.currentTimeMillis() 获取1970 1-1 0:0：0开始走到此刻的毫秒值

// System.exit(0);//终止java虚拟机

Runtime类

exit() 终止虚拟机操作， 一般不用

    '''
        // Runtime.getRuntime().exit(0);  //终止虚拟机
    '''

availableProcessors() 返回虚拟机能调用的处理器数量

    '''
        System.out.println(Runtime.getRuntime().availableProcessors()); //返回虚拟机能使用的处理器数量
    '''

totalMemory() 返回虚拟机中内存总量， long类型 字节

    '''
        System.out.println(Runtime.getRuntime().totalMemory());  //返回虚拟机中内存总量
        System.out.println(Runtime.getRuntime().totalMemory() / 1024.0 / 1024.0 + "MB");
    '''

freeMemory() 返回虚拟机中空闲的内存数量

    '''
        System.out.println(Runtime.getRuntime().freeMemory()/ 1024.0 / 1024.0 + "MB");//返回虚拟机中空闲的内存数量
    '''

exec() 根据路径启动某个程序， 需要管理员权限

    '''
        Runtime.getRuntime().exec("D:\\Star Rail\\launcher.exe");  //启动 某个程序， idea需要管理员权限
    '''

Process.destroy(); //用于结束通过exec启动的程序

    '''
        Process p = Runtime.getRuntime().exec("D:\\Star Rail\\launcher.exe");  //启动 某个程序， idea需要管理员权限, 返回Process类， 即软件对象
        Thread.sleep(5000);  //让程序在此等待5s
        p.destroy();  //结束这个软件
    '''

BigDecimal类， 在java中 浮点类数据计算可能出现精度问题， 用BigDecimal类可以解决

'''
        BigDecimal a = BigDecimal.valueOf(0.1);
        BigDecimal b = BigDecimal.valueOf(0.2);
        System.out.println(a.add(b)); //加
        System.out.println(a.subtract(b));  //减
        System.out.println(a.multiply(b));  // 乘
        System.out.println(a.divide(b));   //除
        System.out.println(b.divide(a, 3, RoundingMode.HALF_UP));   //除数  精确几位， 舍入模式
'''

JDK8新增的时间日期类

LocalDate :代表本地日期（年、月、日、星期）
LocalTime： 代表本地时间（时、分、秒、纳秒）
LocalDateTime：代表本地日期时间：（年、月、日、星期、时、分、秒、纳秒） //用的最多

以LocalDate类为例：

‘’’
LocalDate localDate = LocalDate.now();
System.out.println(localDate);
System.out.println(localDate.getYear());
System.out.println(localDate.getMonth());
System.out.println(localDate.getDayOfMonth());
System.out.println(localDate.getDayOfWeek().getValue());

    LocalDate localDate1 = localDate.withYear(2099); //返回新的localdate类型， 原来的对象数据不变
    System.out.println(localDate1);
    System.out.println(localDate);

    LocalDate localDate2 = localDate.withMonth(12); //返回新的localdate类型， 原来的对象数据不变
    System.out.println(localDate2);

    localDate.plusYears(2);  //加两年， 返回新的对象
    localDate.minusYears(2); //减两年， 返回新的对象

    LocalDate.of(2022, 12, 10);//获取指定时间对象， 返回新的对象
    localDate.equals(localDate1); //判断日期对象是否相等
    localDate.isAfter(localDate1); //判断日期是否在指定对象后面
    localDate.isBefore(localDate1); //判断日期是否在指定对象后面

‘’’

其中， LocalDateTime可以分开也可合并

LocalDate l1 = localDateTime.toLocalDate(); //分开
LocalTime l2 = localDateTime.toLocalTime();

Period类和Duration类

Period类用于计算两个localDAte类相差的年月日
Duration类用于计算两个时间对象相差的天、小时、分、秒、纳秒
用法同上或见API文档

‘’’
LocalDate localDateTime1 = LocalDate.of(2022, 11, 11);
LocalDate localDateTime2 = LocalDate.of(2023, 12, 12);

    Period p = Period.between(localDateTime1, localDateTime2);
    System.out.println(p.getYears());
    System.out.println(p.getMonths());

    LocalDateTime loc1 = LocalDateTime.of(2022, 2, 11, 12, 24, 48);
    LocalDateTime loc2 = LocalDateTime.of(2022, 12, 12, 3, 48, 59);

    Duration duration = Duration.between(loc1, loc2);
    System.out.println(duration.toDays());
